# SecureIT - File Encryption System

Windows-based file encryption system using AES-256 encryption for IT professionals in Fiji.

## Features
- AES-256 encryption
- Secure file deletion
- Password strength validation
- Audit logging
- Batch processing

## Installation
```bash
pip install -r requirements.txt
python main.py
```

## Requirements
- Python 3.11+
- Windows 10/11
- cryptography library
